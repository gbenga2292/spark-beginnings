# Version 1.4.0 Documentation & Changelog

This document summarizes the features and improvements implemented since version 1.3.6.

## 🚀 Core Features & Enhancements

### 1. Integrated Payroll & Attendance System
- **Dynamic Salary Computation**: Implemented `MonthlyPay` logic which adjusts operations staff salary based on actual attendance records vs official workdays.
- **Overtime Engine**: Automated overtime calculation based on Sunday work, holiday shifts, and double shifts (Day/Night).
- **Payslip Generation**:
    - A4-style payslip modal in the Payroll module.
    - Floating action buttons for **Printing** and **Downloading** (as .txt format).
    - Detailed breakdown including Basic Salary, HRA (capped at ₦500k), Transport, Other Allowances, and Deductions (Tax, Pension, Loans).
- **Statutory Compliance**: Integrated `NIGERIATAX` function for Nigerian PAYE tax calculations and 8% Pension deduction logic.

### 2. Salary Advances & Loans Management
- **New Module**: Dedicated "Salary & Loans" page accessible from the sidebar.
- **Salary Advances**: Staff can request advances to be deducted from their net pay at the end of the current month.
- **Structured Loans**: 
    - Support for multi-month durations.
    - Automatic calculation of monthly deductions.
    - Configurable start dates for repayments.
- **Administrative Control**: Approve/Reject workflow for all advance and loan requests with real-time stat tracking (Total Pending, Active Loans, Total Value).

### 3. Financial Reporting & Analytics
- **Live Dashboard**: Real stats cards connected to the global store.
- **Custom Report Builder**: Generates CSV exports for broader auditing.
- **Operations Site Work Reports**: Filterable by Month and Year to track workforce distribution across sites.

### 4. Site & Employee Lifecycle Management
- **Inline Editing**: Quick updates for sites directly on the Sites page.
- **Enhanced Employee Profiles**: Expanded "More" menu with View/Edit/Delete actions and safe deletion confirmation dialogs.
- **Onboarding Flow**: Fully connected to the centralized store for real-time employee data synchronization.

### 5. Billing & Invoicing
- **Unified Billing Hub**: Connected to the global store for consistent invoice tracking.
- **Workflow Actions**: Capabilities to Create, Send, and Delete invoices with dynamic revenue stats calculation.

---

## 🛠️ Technical Improvements
- **Global State Management**: Migrated modules to use `AppDataContext` for unified state across Task management and Financial modules.
- **Route Architecture**: Updated `App.tsx` and `Sidebar` to include new modules (Salary & Loans).
- **Data Persistence**: Improved serialization for Excel-based backups and restores.
